🚀 Getting Started
Prerequisites
Make sure you have the following installed:
bashpip install tensorflow keras opencv-python numpy matplotlib
Installation

Clone the repository:

bashgit clone https://github.com/MohsinMullick/cnn-image-colorization.git
cd cnn-image-colorization

Install dependencies:

bashpip install -r requirements.txt

Open the Jupyter Notebook:

bashjupyter notebook cnn_image_colorization.ipynb

🎯 Project Goal
This project aims to transform grayscale images into realistic colorized images using Convolutional Neural Networks (CNNs). The model learns the mapping from grayscale pixel values to color channels using deep learning techniques.
