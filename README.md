CNN-Based Black and White Image Colorization

A deep learning project that automatically colorizes black and white images using CNN and OpenCV.

 Features
Automatic image colorization
CNN-based deep learning model
Image preprocessing and visualization
Training and prediction pipeline
🛠 Technologies Used
Python
TensorFlow / Keras
OpenCV
NumPy
Matplotlib
 Project Goal

This project aims to transform grayscale images into realistic colorized images using Convolutional Neural Networks (CNNs).
